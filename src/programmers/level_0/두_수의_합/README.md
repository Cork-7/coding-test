# 두 수의 합

### 성능 요약
- 메모리 : 78.4MB
- 시간 : 0.02ms

### 문제 설명
두 정수 `num1`과 `num2`가 주어질 때, `num1`과 `num2`를 더한 값을 return 하도록 solution 함수를 완성해주세요.

---

###  제한 사항
- -50,000 ≤ `num1` ≤ 50,000
- -50,000 ≤ `num2` ≤ 50,000

---

### 입출력 예
| num1 | num2 | result |
| :--- | :--- | :--- |
| 2 | 3 | 5 |
| 100 | 2 | 102 |
